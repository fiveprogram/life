# 20分発表 台本

経営者コミュニティでの20分発表用の台本です。

## File Order

1. [`01_current_business_and_schedule.md`](01_current_business_and_schedule.md): 0:00-1:00 現在の事業と、発表内容のタイムスケジュール
2. [`02_origin_and_values.md`](02_origin_and_values.md): 1:00-5:00 生い立ち・価値観の原点
3. [`03_career_to_founder.md`](03_career_to_founder.md): 5:00-9:00 職歴・経営者になるまで
4. [`04_personal_growth.md`](04_personal_growth.md): 9:00-14:00 人間的に成長したいところ
5. [`05_management_issues_and_questions.md`](05_management_issues_and_questions.md): 14:00-20:00 これからの経営課題と相談

## 使い方

- まずは `[ ]` の中を自分の言葉で埋める
- 各パートで一番伝えたいことを1つに絞る
- 具体例は各パート1つまでにする
- 最後の相談テーマは、参加者が答えやすい問いにする
