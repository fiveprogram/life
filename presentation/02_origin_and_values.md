# 02 生い立ち・価値観の原点

## 時間

1:00-5:00

## 目的

今の経営判断や人との向き合い方につながる、価値観の原点を伝える。

## 台本

次に、私の生い立ちと、価値観の原点について共有させていただきたいと思います。

私は大阪府河内長野市で生まれ育ちました。  
家族は両親と4歳下の弟が1人です。家の雰囲気は、どこかふざけることが好きな両親によって作られていたように思います。母は息子に怒り、父はそれを優しく見守る。そんな役割分担のようなものが、我が家にはありました。

幼い頃から、周りには「つかみどころのない子供だね」と言われていました。  
自分の好きなものや好きな人には自然と近づいていく一方で、一般的な子供が好きそうなものにはあまり興味を示さない。周りの大人からすると、少しコントロールしづらい子供だったのかもしれません。

小学生の頃は、とにかく楽しいことに一直線でした。  
学校から帰ると友達の家に向かい、目の前にある楽しいことに夢中になる。勉強は二の次で、サッカー、ピアノ、テニス、そろばん、スイミング、塾など、いろいろな習い事をさせてもらっていました。

両親からの期待に応えなければいけないというプレッシャーがあり、最初は結果と向き合うのが怖くて逃げてしまう自分もいました。  
それでも最後までやり切ると決めて、2ヶ月後に合格することができました。逃げたい気持ちを抱えながらも、最後までやり切った初めての経験だったと思います。

一方で、自分の性格の原点になっている大きな出来事は、中学1年生の時にあります。  
親友5人と彼女が、自分から離れていってしまいました。

今振り返ると、周りの精神年齢が上がっていく中で、自分の他者への振る舞いの幼さが表に出てしまったのだと思います。  
小学生の頃には許されていたことが、中学生になると許されなくなる。その変化に、自分はうまく対応できていませんでした。  
この出来事から、人に対する怖さを持つようになった一方で、他者には自分からgiveしなければいけない、という気持ちも芽生えました。

中学時代には、テニス部での経験も大きかったです。  
長期休みには1日8時間くらい練習するような厳しい部活でした。大会で大事な試合を落としてしまい、部活での立ち位置が一気に下がったことがあります。かなり悔しい経験でした。
ただ、その挫折から、練習への向き合い方が変わりました。  
誰よりも声を出し、誰よりも集中して練習するようになりました。結果的に、1ヶ月後の試合では、また一番強いメンバーとペアを組んでダブルスに出ることができました。  
この経験から、追い込まれた時に自分で立て直す力や、負けたくない気持ちは強くなったと思います。

少し性格診断のような話もすると、私を表すものとして、MBTIではENTJ、六星占術では水星人プラス、ソウルナンバーは1、動物占いでは束縛を嫌う黒豹という結果があります。
もちろん、こうした診断をすべてそのまま信じているというより、自分の性格を言語化する材料として見ると、かなり納得する部分があります。

共通しているのは、自分で決めたい、前に進めたい、誰かに管理されるよりも自分の判断で動きたい、という傾向です。  
ENTJという面では、物事を感情だけで見るよりも、目標や成果、全体の構造で考えることが多いです。ソウルナンバー1というところも、自分で道を作りたい、先頭に立って進めたいという感覚に近いと思っています。  
一方で、水星人プラスや束縛を嫌う黒豹という要素は、自由でいたい、自分のペースを大事にしたい、納得できないルールには違和感を持ちやすい、というところに出ている気がします。

この性格や経験は、経営者としては強みになっています。  
意思決定を先送りせず、まず方向を決める。課題があれば構造化して、どうすれば前に進むかを考える。人に依存しすぎず、自分で責任を持って動く。そういう部分は、今の仕事にもかなり活きています。

ただ、裏返しもあります。  
自分の中で答えが見えた時に、周りの感情やペースを十分に待たずに進めてしまうことがあります。自由を大事にする分、細かく管理されたり、納得感のない指示を受けたりすると、強く反発してしまうこともあります。  
また、人間関係で一度大きく失敗した経験があるからこそ、人との距離感に慎重になりすぎる部分もあります。

今振り返ると、私の価値観の原点は、自由と責任をセットで持ちたいというところにあると思います。  
誰かに決められた道を歩くより、自分で選び、その結果を自分で引き受ける。その感覚が、仕事観や経営スタイルの土台になっています。

なので、私にとって生い立ちや性格の話は、単なる自己紹介ではありません。  
自分がなぜ経営者という立場を選んだのか、なぜ意思決定やスピードを大事にするのか、そしてなぜ人との関わり方をもっと成長させたいと思っているのかを説明する、重要な背景だと思っています。
